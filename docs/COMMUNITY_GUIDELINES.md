# 커뮤니티·장소명 가이드라인 (초안)

> **이 문서는 초안이다.** 운영자는 실제 신고가 들어오는 패턴을 보면서 보완할 것.

서비스의 가치는 "내가 지금 있는 장소의 분위기를 솔직히 한 표 던질 수 있다"는 점에 있다. 장소 데이터가 깨끗해야 그 가치가 유지된다.

## 1. 허용되는 장소

- **공개 공간**: 강의실, 지하철 열차/역, 카페, 버스 노선, 도서관, 사무실 공용 공간 등 다수가 함께 머무는 장소.
- **식별 가능한 단위**: "서울대학교 301동 401호", "2호선 강남→삼성 열차", "스타벅스 강남대로점" 처럼 명확히 한 장소를 가리키는 이름.
- **일관된 형식**: 같은 종류의 장소는 같은 분류(`type`)와 비슷한 표기 방식을 따른다.

## 2. 허용되지 않는 장소·표현

다음에 해당하는 장소·이름은 **자동 거절되거나 운영자 판단으로 삭제**될 수 있습니다.

### 입력 검증으로 자동 거절

- URL(`http`, `https`, `www.`) 포함
- 한국 휴대전화 번호(`010-XXXX-XXXX`)·일반전화 번호 형태 포함
- "카톡", "오픈채팅", "텔레그램", "무료 상담" 등 영업·홍보 의도 키워드
- 동일 문자 7회 이상 반복(`ㅋㅋㅋㅋㅋㅋㅋ`, `!!!!!!!!` 등)
- 2자 미만 또는 40자 초과 (정상 장소명은 이 범위로 충분)

### 사후 신고·운영자 판단으로 삭제

- 본인 또는 제3자의 개인정보(전화번호·주소·계좌·민감정보 등)
- 욕설, 혐오 표현, 차별, 음란, 폭력 묘사
- 정치 선동, 특정 인물·집단·기관에 대한 명예훼손성 표현
- 광고, 영업, 홍보, 후원 모집
- 실재하지 않는 장소, 식별 불가능한 모호한 이름("아무데나", "여기")
- 동일 장소의 의도적 다중 등록 (병합 대상)
- 운영자가 부적절하다고 합리적으로 판단하는 기타 콘텐츠

## 3. 투표 가이드

- 한 장소에 한 표만 던집니다. 의견이 바뀌면 30초 후 변경할 수 있고, 1시간 뒤 자동 만료됩니다.
- 같은 장소에 여러 번 투표하려고 쿠키를 지우거나 시크릿 모드를 사용하면, 시스템이 동일 IP 대역으로 인식해 추가 표를 차단할 수 있습니다.

## 4. 신고

부적절한 장소·표현을 발견하면 다음으로 신고해주세요.

- 이메일: TODO: report@aircondemocracy.com
- 신고 시 포함하면 좋을 정보: 장소 URL(`/p/<id>`) 또는 이름, 신고 사유, 발견 시각

운영자는 신고를 접수하면 **24~72시간 이내**에 검토하고, 필요한 경우 비공개·삭제·차단 조치를 합니다.

## 5. 차단 정책

- **1차 위반**: 해당 투표/장소 삭제, voter_id 1~24시간 신규 투표 차단.
- **반복 위반**: voter_id 또는 IP 대역 7~30일 차단.
- **심각/조직적 어뷰즈**: 영구 차단, 필요 시 수사기관 협조.

차단은 voter_id 해시 또는 IP prefix 해시로 적용되며, 사용자에게 차단 사유와 해제 시점은 별도 통지하지 않습니다(차단 우회 학습 방지). 본인이 부당하게 차단되었다고 판단되는 경우 위 이메일로 이의 제기할 수 있습니다.
