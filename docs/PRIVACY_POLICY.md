# 개인정보처리방침 (초안)

> **이 문서는 초안이다.** 실제 게시 전 반드시:
> 1. 운영자 연락처 / 처리책임자 / 사업자등록정보 채우기
> 2. PIPA(개인정보 보호법) 최신 가이드 또는 변호사 검토
> 3. 카카오 OAuth scope 최종 확정에 맞춰 수집항목 갱신
>

"에어컨 민주주의"(이하 "서비스")는 사용자의 개인정보를 중요하게 다루며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 처리방침은 서비스가 처리하는 개인정보의 항목, 목적, 보유기간, 보호조치 등을 알려드립니다.

## 1. 처리 항목과 수집 방법

| 구분 | 항목 | 수집 방법 |
|---|---|---|
| 익명 이용자 | 익명 voter_id (서명된 임의 UUID), 쿠키 | 최초 접속 시 자동 발급 |
| 익명 이용자 (남용 방지) | IP 주소의 일 단위 HMAC 해시 (IPv4 `/24`, IPv6 `/64`), User-Agent 해시 | 요청 헤더에서 추출 후 즉시 해시화하여 저장 |
| 카카오 로그인 이용자 | 카카오 회원번호, 닉네임, 프로필 이미지 URL, 이메일(선택) | 카카오 OAuth 인증 시 |
| 투표 데이터 | 장소 ID, 투표(시원/적당/더움), 투표·변경 시각, 만료 시각 | 이용자가 직접 입력 |
| 장소 등록 | 장소 이름, 분류, 지역, 상세, 작성자(voter_id 또는 user_id) | 이용자가 직접 입력 |

서비스는 **raw IP 주소를 영구 저장하지 않습니다.** Cloudflare(엣지)에서 처리되는 일시적인 요청 메타데이터를 제외하고, 앱 데이터베이스에는 IP 주소가 HMAC 해시 형태로만 보존되며 일자가 바뀌면 같은 IP라도 다른 해시 값이 됩니다.

## 2. 처리 목적

- **서비스 제공**: 장소별 익명 투표 집계, 결과 노출, 새 장소 등록.
- **남용 방지**: 동일 사용자/IP의 과도한 투표·장소 생성 차단, 자동화 공격 탐지(`docs/ABUSE_PREVENTION_POLICY.md` 참고).
- **보안 감사**: 비정상 트래픽 사후 분석, 사고 대응.
- **카카오 로그인**: 본인 식별, 로그인 상태 유지.

법령 또는 정부 기관의 적법한 요청이 없는 한, 마케팅·광고·프로파일링 목적으로 사용하지 않습니다.

## 3. 보유 및 이용 기간

| 항목 | 보유 기간 |
|---|---|
| 익명 voter_id 쿠키 | 발급일로부터 180일 또는 이용자가 쿠키 삭제 시 |
| 투표 (`votes`) | 만료(통상 1시간) 후 1~2시간 내 삭제 |
| 감사 로그 (`audit_events`) | 30일 |
| Rate limit 버킷 (`rate_limit_buckets`) | 윈도우 종료 후 최대 24시간 |
| 차단 대상 (`blocked_subjects`) | 사유별 1시간~30일 |
| 장소 (`places`) | 운영상 필요한 기간 동안 보유. 부적절·개인정보 포함 시 즉시 삭제 |
| `places.created_by` (voter_id/user_id) | 30~90일 후 null 처리 권장 |
| 카카오 로그인 회원 정보 | 회원 탈퇴 시 즉시 삭제. 1년 이상 미사용 시 삭제 또는 비활성화 |

## 4. 제3자 제공 및 처리 위탁

서비스는 개인정보를 제3자에게 제공하지 않습니다. 다만 아래 서비스에 처리를 위탁합니다.

| 수탁자 | 위탁 업무 | 처리 위치 |
|---|---|---|
| Cloudflare, Inc. | 호스팅(Workers/Pages/D1), 보안(WAF, Rate Limiting), CDN | 글로벌 (국외 처리 발생 가능) |
| 카카오 (Kakao Corp.) | OAuth 로그인 인증 | 대한민국 |

Cloudflare는 엣지에서 IP 주소 등의 요청 메타데이터를 일시적으로 처리합니다. 자세한 사항은 Cloudflare 개인정보처리방침을 참고하세요.

## 5. 국외 이전

이용자의 요청 메타데이터(IP 주소, User-Agent 등)는 Cloudflare 글로벌 네트워크에서 처리되며, 가장 가까운 엣지 노드에서 응답이 생성됩니다. 이로 인해 일시적 국외 이전이 발생할 수 있습니다.

| 이전 받는 자 | 이전 국가 | 이전 항목 | 이전 일시·방법 | 보유·이용 기간 |
|---|---|---|---|---|
| Cloudflare, Inc. | 미국 및 글로벌 엣지 | IP 주소, User-Agent, 요청 URL | 요청 발생 시 네트워크 전송 | Cloudflare 정책에 따름 |

## 6. 정보주체의 권리

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

- **열람**: 본인의 카카오 로그인 정보 확인 (서비스 내 마이페이지 또는 이메일 요청)
- **삭제**: 카카오 로그인 회원 탈퇴, 또는 익명 쿠키 초기화
- **처리정지**: 이메일 요청 시 신규 데이터 수집 중단

요청 처리는 접수일로부터 **10일 이내**를 목표로 처리합니다. 익명 투표 데이터는 voter_id 쿠키 외에 본인 식별 수단이 없어 사후 식별·삭제가 제한적임을 양해 부탁드립니다(투표 데이터는 1시간 후 자동 삭제).

## 7. 안전성 확보 조치

- **접근통제**: 운영용 D1 콘솔은 운영자 1인만 접근. Cloudflare 계정은 2FA 강제.
- **암호화**: HTTPS 전송 강제. 쿠키·세션 토큰은 HMAC 서명. IP·voter_id·UA는 모두 HMAC-SHA256 해시로만 저장.
- **접근 기록 보관**: `audit_events`에 30일 보관 후 자동 삭제. raw 식별자는 기록하지 않음.
- **최소 수집**: 카카오 OAuth scope에서 이메일은 선택, 닉네임·프로필 이미지는 표시용으로만 사용.
- **외부 SaaS 미사용**: 별도 외부 분석/로그 SaaS로 데이터를 전송하지 않음.

## 8. 자동 수집 도구

서비스는 다음 자동 수집 도구를 사용합니다.

- **익명 식별 쿠키 (`voter`)**: 남용 방지를 위한 익명 식별자. 거부 가능하나 거부 시 투표가 정상 동작하지 않을 수 있습니다.
- **세션 쿠키 (`session`)**: 카카오 로그인 상태 유지. 로그아웃 시 즉시 삭제.

별도의 외부 분석 도구(Google Analytics 등)는 사용하지 않습니다.
