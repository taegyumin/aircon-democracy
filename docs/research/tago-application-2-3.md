## 신청 #2 — TAGO 15098522 고속버스정보

신청 URL: https://www.data.go.kr/data/15098522/openapi.do

### 서비스 설명

```
'에어컨 민주주의(aircondemocracy.com)'는 대중교통(지하철·버스·간선철도) 차량의 냉방 상태에 대해 시민이 익명으로 '춥다/적당/덥다'를 투표하고, 같은 차량 다른 승객의 투표 집계를 실시간으로 확인할 수 있는 비상업·무광고 시민 참여 웹서비스입니다.

대상: 한국 대중교통 일반 이용자 (로그인 불필요).
목적: 차량별 객관적 냉방 만족도 데이터를 모아 운영기관·승객 모두에게 가시화.

차량 단위 식별을 위해 다음 공공데이터를 이미 활용 중입니다:
- 서울 swopenAPI (수도권 지하철 실시간 차량 위치)
- TAGO 버스위치정보(1613000, 15098533) — 전국 시내버스 vehId

이번 신청 건(국토교통부 TAGO 고속버스정보)은 KOBUS·센트럴 등 전국 고속버스 차량 투표를 추가하기 위해 필요합니다. 고속버스는 승차권에 출발 터미널·도착 터미널·출발 시각이 명시되므로 사용자가 입력한 출도착지·시각이 실제 배차에 존재하는지·운행 시간대 안인지 시각표 API로 검증하여, 유효한 차량에만 투표가 기록되도록 합니다.
```

### 기능 및 서비스 화면

```
고속버스 투표 흐름에서 본 API를 다음과 같이 사용합니다:

1. 출발·도착 터미널 자동완성:
   - getExpBusTrminlList 등으로 전국 고속버스 터미널 목록 로드
   - 사용자 도시 선택 → 해당 도시의 터미널만 노출
2. 버스 등급 라벨링:
   - getExpBusGradeList 등으로 일반/우등/프리미엄 등 enum 로드, UI에 노출
3. 배차 검증 (핵심):
   - 사용자가 입력한 출발일·출발 터미널·도착 터미널·출발시각·버스 등급으로
     getStrtpntAlocFndExpbusInfo 호출 → 응답에서 입력값과 일치하는 row 검색
   - 일치하는 배차가 있고 현재 시각이 출발~도착 사이일 때만 투표 페이지로 진행
   - 일치 없거나 운행 외 시간이면 안내 후 입력 화면 복귀

투표 자체는 자체 서버 D1에 익명 저장되며, 본 API는 검증·정보 조회 용도로만 호출하고 사용자 PII는 일체 전송하지 않습니다. 호출 빈도는 사용자가 버스 입력 단계를 거칠 때마다 1~3건 수준입니다.
```
