# Missing Schools Result

2026-05-27 기준으로 후보 6개를 각각 검색하고, 건물 5개 이상이 서로 다른 도메인의 공개 출처에서 맞는 경우만 데이터에 반영했다.

| 후보 | 위치 | 시도 결과 |
|---|---|---|
| 단국대 천안캠퍼스 | 충남 천안 | 성공. 기존 `dankook-cheonan`을 4건물에서 8건물로 보강했다. 공식 천안캠퍼스/편의시설 페이지와 K-위키·우만위키 계열 페이지에서 인문과학관, 사회과학관, 의학관, 생명자원과학관, 학생회관 등이 교차 확인됐다. |

| 협성대학교 | 경기 화성 | 성공. `hyupsung`을 8건물로 추가했다. 공식 캠퍼스 투어/입시 PDF와 Unisquads 공개 문서에서 본관, 중앙도서관, 인문사회관, 이공관, 학생회관 등이 교차 확인됐다. |
