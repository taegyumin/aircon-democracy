## 주요 발견

### A. 동번호 체계가 잘 정립된 학교 (★★★★+)

호실 단위까지 가지 않더라도 건물 검색이 곧장 가능한 학교들. 이들은 **연세대-like UX(건물 검색 + 호실 freeform)**로 빠르게 추가 가능:

1. **동국대** — 1~36 (1명진관·2본관·3만해관·4법학·5정각원·6혜화·7혜화별관·8경영·9사회과학·10문화·11학술·12박물관·13다향·14금강·15체육·16학림·17·18계산AB·19학생회관·20·21정보문화QP·22·23원흥·24원흥별관·25남산학사·26신공학·27과학·28중앙도서관·29상록원/충무로영상센터·30반야·31명심) — **번호 + 이름 매핑 거의 완비**
2. **한양대 서울** — 9개 zone (101~909), 100단위 구역제. 위키에 전체 매핑 있음
3. **국민대** — N/S/E/W + 번호 (N1북악·N2본부·W1공학·W2성곡도서관·E1생활관 등)
4. **숭실대** — 01~11+ (01베어드홀·02숭덕경상·03문화·08한경직·09신양·10벤처·11진리)
5. **홍익대 서울** — A~V 알파벳 (가온A·강당S·공학CD·과학I·국제교육V·남문W)
6. **중앙대 서울** — 100/200/300 zone (301중앙문화예술·302대학원·303법학·304미디어공연영상·305교수연구·307~309기숙사·310 100주년기념)
7. **한국외대** — 0~12 단순 번호 (0본관·1인문·2교수학습·3사회과학·4학생회관기숙사·5법학·6대학원·7외국어연수평가·8국제·9도서관·10교수연구·11교수회관·12출판)
8. **서울시립대** — 동번호 (1전농관 등)
9. **서울과기대** — 동번호 + 명칭, 공식 인터랙티브 맵
10. **고려대 안암** — 공식 캠퍼스맵에 동번호 체계 있음

### B. 인증 필요 — 공식 호실 시스템이 있으나 로그인 막힘

- **성균관대 강의종합안내도** (`classinfo.skku.edu`) — 로그인 강제. 학생들이 실제 강의실 찾을 때 쓰는 정식 시스템. 이게 뚫리면 SNU급 데이터 확보 가능.
- 다른 학교들도 학사포털 내부에 비슷한 시스템 있을 가능성 — 다 인증벽.

### C. 캠퍼스 작아 자유 입력으로 충분한 학교

서울교대, 한성대, 가톨릭대 혜화, 덕성여대, 상명대 서울, 서경대, 감리교신학대, 장신대, 서울기독대, 한국성서대, 추계예술대 — 학생 수 적고 건물 10~20개 이내. UI 다 만들기 비용 > 효용.

### D. 특수 케이스

- **KAIST 서울 (홍릉)** — KAIST 본 캠퍼스는 대전이고 서울은 경영대학원 위주. "호관" 번호. 우선순위 낮음.
- **육군사관학교** — 군 시설이라 데이터 공개 제한. 생도/장교만 사용 가능 + 보안. **제외 권장**.
- **한국예술종합학교** — 멀티 캠퍼스(석관·서초·대학로) 분리 필요. 학과 단위로 위치 다름.
- **한국방송통신대** — 본부 캠퍼스 외엔 학생 출석 거의 없음 (방송 강의). 강의실 voting use case가 모호.
